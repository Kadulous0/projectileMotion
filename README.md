# projectileMotion
2-D customizable simulation of atmospheric re-entry and decay of Earth's atmosphere. Features a custom implementation of the 1976 Standard Atmosphere model along with capsule data from the Apollo Command Module (CM) and in the future, the Orion CM.

## Work In Progress Features
- Multiple Types of Capsules
- Body Lift
- Improvements to Capsule Data
- Improvements to Atmospheric Data
- Improving Simulation Performance

## Future Plans
- Parachute Deployments
- Other Planets
- Sun/Moon Spacecraft Dynamics
- Weather
- 3D Implementation
