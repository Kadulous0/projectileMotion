# projectileMotion
2-D customizable simulation of atmospheric re-entry into Earths (maybe others too in the future) atmosphere. Using a custom implementation of the 1976 Standard Atmosphere model along with capsule data from the Apollo Command Module (CM) and Orion CM in the future.

## Work In Progress Features
- Multiple Types of Capsules
- Body Lift
- Improvements to Capsule Data
- Improvements to Atmospheric Data
- Improving Simulation Performance

## Future Plans
- Parachute Deployments
- Other Planets
- Sun/Moon Spacecraft Dynamics
- Weather
- 3D Implementation
